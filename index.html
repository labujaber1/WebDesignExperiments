Design experiments

A space to experiment with page and component designs influenced by new trends, old ideas, and general messing around.

This page contains links to other pages containing page designs and grouped ideas but also displays unique component ideas (for 2 minutes at least). Experiments are done using custom css and as little javascript as possible whilst the main layout uses bootstrap 5. Things to consider, everyone including this page uses flat layout due to the production line that is the working development process. So whats next?
